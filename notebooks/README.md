# Notebooks

The notebooks in `notebooks/exploratory/` preserve the original lesson-style
experiments. Reusable code has been extracted into `src/efficient_llm_serving`
so production logic can be tested, benchmarked, and reused outside Jupyter.

Some notebooks came from an external instructional environment and originally
referenced helper modules that were not committed to this repository. Prefer
the package modules and examples for reproducible execution.
